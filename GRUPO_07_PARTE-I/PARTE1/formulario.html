FORMULARIOS

Gracias a el DOM y Javascript podemos manipular y validar los campos de los formularios HTML.

-Validación de formularios
Se puede validar si los campos están vacíos y notificar mediante un mensaje.
function validarForm() {
let x = document.forms["miForm"]["fname"].value;
if (x == "") {
alert("Name must be filled out");
return false;
} }

-Validar entrada numérica
Valida la entrada de números de un campo
function myFunction() {
let x = document.getElementById("numb").value;
let text;
if (isNaN(x) || x < 1 || x > 10) {
text = "Input not valid";
} else {
text = "Input OK";
}
document.getElementById("demo").innerHTML = text;
}
