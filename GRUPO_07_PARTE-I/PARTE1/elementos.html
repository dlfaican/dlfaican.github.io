ELEMENTOS

Con Javascript se desea manipular elementos de HTML los cuales podemos acceder de diferentes maneras como:

-ID
Se utiliza el ID del elemento que queremos encontrar. Como en el ejemplo:
const elemento = document.getElementById("idElemeto");

-Nombre de etiqueta
Encuentra todos los elementos con esa etiqueta. Ejemplo:
const elemento = document.getElementsByTagName("li");

-Nombre de clase
Encuentra todos los elementos con esa clase. Ejemplo:
const x = document.getElementsByClassName("txt-color");

-Selectores CSS
Encuentra todos los elementos con la clase y seguido su etiqueta. Se utiliza el método querySelectorAll()
const x = document.querySelectorAll("li.txt-color");

-Colecciones de objetos HTML
Esta llamada se usa generalmente cuando existen más elementos en ese objeto, como por ejemplo en un formulario que te obtiene todos los elementos que están dentro del formulario. Ejemplo:
const x = document.forms["form1"];
