DOM HTML

El HTML DOM nos permite cambiar el contenido del HTML haciendo uso de Javascript

-Cambiar el contenido de un elemento
Para hacer este proceso usamos la propiedad de innerHTML que nos permite cambiar el contenido de un elemento. Ejemplo:
Es decir si tenemos un parrafo con el id=parrafo. Para cambiar el texto dentro de ese párrafo sería así: document.getElementById("parrafo").innerHTML = "Aqui se ingresa el nuevo contenido";

-Cambiar el valor de un atributo
Para el cambio de un atributo primero encontramos el elemento y seguido un punto con el nombre del atributo. La sintaxis queda asi: document.getElementById(id).attribute= nuevoValor. Ejemplo del cambio de href de un ancla:
document.getElementById("ancla").href = "#home";

-Contenido HTML Dinámico
Gracias a javascript podemos crear contenido HTML dinámico. Ejemplo:
document.getElementById("conteiner").innerHTML = "Fecha : " + Date();
